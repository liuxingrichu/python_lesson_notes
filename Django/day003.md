8. 模板渲染
	- 特殊的模板语言
	###
		def func(request):
			return render(request, 'index.html', 
				{"current_user": "Look", 
				"user_list": ['one', 'two'],
				"user_dict": {"k1": 'v1', "k2": 'v2'}，
				"age":10})

		index.html
		<html>
			<body>
				# 变量
				<div>{{ current_user}}</div>

				# 列表
				<a>{{ user_list.0 }}</a>
				<a>{{ user_list.1 }}</a>
				
				# 字典
				<a>{{ user_dict.k1 }}</a>
				<a>{{ user_dict.k2 }}</a>
			
				# 条件语句，可嵌套
				{% if age %}
					<a>有年龄</a>
						{% if age > 16 %}
							<a>老男人</a>
						{% else %}
							<a>小鲜肉</a>
						{% endif %}
				{% else %}
					<a>无年龄</a>
				{% endif %}
				
				# for循环与条件语句混合使用
				{% for row in user_list %}
					{% if row == "spring" %}
						<a>{{ row }}</a>
					{% endif %}
				{% endfor %}

			</body>
		</html>

	- 渲染时间：返回给用户数据之前
	- 效果：用户拿到的数据，都是渲染后的
